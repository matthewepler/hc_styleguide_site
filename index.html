HC Web Dev Guide

INTRO

This is how we make the interwebs at HC. Everything you see is built with stuff we actually use. When making a new web project, you can use this as your guide for all aesthetic and technical choices. this is meant to be a living document. as our standards change, so shall this site. Amen.

HOW TO USE THIS SITE

This site is meant as a quick reference with links to the tech we use and explanations of how we use them. Use it to figure out what you should use so we're all in sync. This site is also used to generate our style guide with this doohickey. For our complete style guide, go here.

Browsers and Devices

Supported browsers

As of 2016, we support the latest version of every browser as well as the previous version.

IE Edge Firefox Chrome Safari Opera iOS Safari Opera Mini Android Browser Chrome for Android
9 13 45 49 9 36 8.4 8 4.4.4 50

Supported Operating Systems

At this time, we develop for the most current operating system across platforms. If a project is going to be released to the market, we rely on third-party vendors to take care of compatibility issues.

Programming for Supported Devices & Hybrid Apps

Because of our existing partnerships, we try to compile for iOS, Androids, and Windows mobile platforms. This is done using the technology of your choice. As of Spring, 2016, the Product team is using React Native, and R&D has explored Intel XDK and is learning React but no platform has been chosen.
